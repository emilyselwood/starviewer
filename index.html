Orb Viewer

This is a model of the local volume with the 1000 nearest stars marked. Each point shows the real position of a star.

Left click and drag to look around. Right click and drag to move around. Scroll to zoom in and out.

There is a 'T' button in the top right next to the help button that will enable a flight through the solar system.

The data is sourced from Robert McNees and processed and visualised by Emily Selwood and collaborators using Orb Calc to give positions. This project is open source and can be found on github
